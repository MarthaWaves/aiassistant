# ИИ-ассистент по agile-планированию

## О проекте
Проект представляет собой интеллектуального ассистента, созданного для помощи в эффективном планировании времени и достижении целей с использованием методологии Agile. Он основан на принципах из книги Катерины Ленгольд "Agile Life" и помогает пользователям структурировать свои цели, распределять время, следить за балансом энергии и поддерживать мотивацию.

## Функционал
ИИ-ассистент обладает следующими ключевыми возможностями:

- **Осмысленное ведение диалога**: Позволяет пользователю взаимодействовать через естественный разговорный язык, предоставляя профессиональные рекомендации по управлению временем и энергией.
- **Ответы на вопросы по базе знаний**: Отвечает на конкретные вопросы о тайм-менеджменте, привычках, выгорании и других аспектах продуктивности, основываясь на содержании книги "Agile Life".
- **Добавление задач в Google Календарь**: Интегрируется с календарем Google для автоматического добавления задач и событий, что упрощает процесс планирования.
- **Рекомендации по спринтам**: Помогает выбирать правильные цели для каждого спринта (3 недели), анализировать прогресс и корректировать планы.

## Технологии и сервисы
Для реализации проекта использовались следующие технологии:

- **Qwen**: Нейросеть Qwen была задействована для создания системного промпта и формирования базы знаний, которая лежит в основе работы ассистента.
- **Savvy (https://suvvy.ai/)**: Сервис для создания и управления ИИ-ассистентами, который позволил легко интегрировать функционал и настроить интерфейс взаимодействия.

## Особенности проекта
- **Гибкость**: Ассистент адаптируется к текущему состоянию пользователя, используя концепцию Agile Compass для анализа трёх сфер жизни — личной, социальной и карьерной.
- **Фокус на восстановлении**: Учитывает важность восстановления энергии (тела, разума, смысла) и предлагает конкретные шаги для её поддержания.
- **Программирование привычек**: Помогает создавать полезные привычки с использованием формулы "Что-Где-Когда-Как" и микронаград для закрепления изменений.
- **Цикл PEAR**: Реализует концепцию цикла Plan-Execute-Analyze-Relax для регулярного планирования, выполнения задач, анализа результатов и отдыха.

## Пример использования
1. **Постановка целей**:
   - Пользователь определяет свои приоритеты в каждой сфере жизни.
   - Ассистент помогает сформулировать реалистичные цели для спринта и разбивает их на помидорные блоки.
2. **Анализ прогресса**:
   - Еженедельно проводится рефлексия, где фиксируются достижения, выявляются трудности и предлагаются шаги для улучшения.
3. **Восстановление**:
   - Рекомендуются практики для восстановления энергии, такие как медитация, digital detox и интервальные тренировки.

## Как это работает
Пользователь может задавать вопросы или получать рекомендации по различным аспектам планирования и самокоучинга. На основе анализа текущего состояния пользователя (через Agile Compass) ассистент предоставляет персонализированные советы и помогает составить план действий.

### Примеры вопросов:
- Как правильно выбрать цели для спринта?
- Что делать, если чувствую выгорание?
- Как внедрить новую привычку без стресса?
- Как оптимизировать своё расписание?

## Интеграции
- **Google Calendar API**: Для автоматического добавления задач и событий в календарь пользователя.
- **Markdown-форматирование**: Для удобного представления информации и сохранения её в чёткой структуре.

## База знаний
База знаний была создана на основе книги "Agile Life" и включает:
- Механизмы работы мозга (префронтальная кора, амигдала, стриатум).
- Стратегии восстановления энергии.
- Принципы управления задачами через технику Pomodoro.
- Методы программирования привычек.
- Концепцию гибкого планирования через спринты.

## Заключение
Проект демонстрирует возможность создания персонального помощника по тайм-менеджменту и самокоучингу с использованием современных технологий искусственного интеллекта. Это решение помогает людям не только становиться более продуктивными, но и поддерживать здоровый баланс между работой и личной жизнью.
